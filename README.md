# Order Form Interface Project 

## Tech Stack

- Vue 3 (Composition API)
- Vuex
- Tailwind CSS
- Axios
- [Maska](https://beholdr.github.io/maska)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
